## INTRODUCTION

This will provide Mega Menu for Provus Site.

## FEATURES

- Create Mega Menu for the site
- Ability to add image and link on the right side of the mega menu

## REQUIREMENTS

- This module requires menu_link_extras: https://www.drupal.org/project/menu_item_extras

## INSTALLATION

- This is now installed via recipe. Just run the install-provus.sh and use gov subinstaller. You could use this command `bash install-provus.sh`

## CONFIGURATION

- Add Menu on you main menu as you normally would.
- Incase you need to add image and link on parent menu that would be located on the right side of menu, edit the parent menu then add Callback image and Callback link

## FRONT END

- Vist the site and verify the mega menu.
